# git commit 关闭eslint校验

// git commit 提交报错，例如这种
xxxx@1.0.0 lint
> eslint src/**/*.{ts,js,vue} --fix

TIP

vue项目中带eslint校验，在提交代码时遇到报错。取消校验即可，方法如下：

解决办法

  • 方法一
git commit -m "提交信息" --no-verify
  • 方法二 在项目里面新建.eslintignore文件，在里面忽略就行，例如忽略src/components文件夹

项目目录：

忽略代码：

 *.sh
node_modules
*.md
*.woff
*.ttf
.vscode
.idea
dist
/public
/docs
.husky
.local
/bin
.eslintrc.js
prettier.config.js
src/assets
src/components
